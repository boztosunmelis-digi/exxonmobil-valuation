# Code of Conduct

## Purpose

This repository exists to share **analytical, educational, and research-grade**
financial modelling work in a respectful and professional environment.

All contributors, reviewers, and users are expected to engage constructively
and in good faith.

---

## Expected Behaviour

Participants are expected to:

- Communicate respectfully and professionally
- Focus on ideas, models, assumptions, and implementation; not individuals
- Accept constructive feedback and differing analytical views
- Credit sources and prior work appropriately

---

## Unacceptable Behaviour

The following behaviour is not tolerated:

- Harassment, personal attacks, or discriminatory language
- Misrepresentation of othersâ€™ work or intent
- Bad-faith criticism without analytical substance
- Spam, trolling, or promotional abuse

---

## Scope

This Code of Conduct applies to:

- GitHub issues
- Pull requests
- Discussions
- Any direct interactions related to this repository

---

## Enforcement

Violations may result in:

- Comment removal
- Issue or pull request closure
- Temporary or permanent restriction from participation

Enforcement decisions are made at the discretion of the repository maintainer.

---

## Reporting Concerns

If you experience or observe unacceptable behaviour, please report it
privately via GitHub messaging to the repository maintainer.

All reports will be reviewed fairly and discreetly.
