# Valuation Models

This folder contains the Excel valuation models associated with the ExxonMobil 5-year DCF, WACC build, and illustrative LBO scenario.

Files included:

- `dcf/DCF_ExxonMobil.xlsx` – Core discounted cash flow model
- `lbo/LBO_ExxonInfrastructure.xlsx` – Infrastructure-style leveraged buyout model

Excel files are placeholders until financial inputs and assumptions are populated.
